# 淘梦获9000万B轮融资，解读这位网生市场优质玩家如何一路顺流而上

*刘卉|2017-11-13|大公司 
                                                创投*

![Image](http://si1.go2yd.com/get-image/0IHbBiTLssi)

网大不行了？其实这个领域才刚刚开始。

“我们B轮已经完成了，额度在九千万，B+轮也很快就会完成。”淘梦创始人&CEO阴超告诉娱乐资本论（ID：yulezibenlun），此次淘梦B轮融资由华映资本、赛富动势基金、德同资本三家共同领投，而且淘梦的B+轮也即将完成，两轮融资总金额在一亿以上。

这家成立于2012年，以做众筹起家的互联网公司如今已是中国网生内容尤其是网大市场中的资深玩家，从参与网大宣发到开始主投主控项目，可以说淘梦是与中国网络大电影一起成长共同成熟，在今年网大进入良性循环阶段，淘梦也在日趋健康的市场环境中再度获得融资，足可见，高手不仅是逆流而上，顺流更要上。

1.互联网思维做产品，淘梦打造市场化数字内容创新平台

以产品化思维去运营业务，以工业化体系制定制片流程，通过数据驱动指导创意开发，通过播后跟踪系统升级再生产是淘梦的核心竞争力，如今的淘梦正致力于将自身打造成为一个集开发、制作、宣发、衍生等多功能为一体的内容平台，内容平台中又会细分出重要环节，如前期策划、内容生产、宣传发行等。产品上则涵盖了网大、网剧、网络文学、短视频、游戏等。

目前我国网民数量已经达七亿多，互联网普及率达到53.2%，在如此庞大的数字基础上，数字内容是中国目前及未来最大的产业之一，看准了这样的市场机会，淘梦也正在搭建一个属于自己的内容平台，今年淘梦提出打造数字内容创新平台的概念，并以C2F（观众对生产者）的工业化体系运作，对于这个概念阴超解释说：“数字内容的定义就是互联网+文化产业”。

数字内容创新平台基于“数据+人才”基础之上，一手研发淘梦大数据机制，一手打造淘梦的人才计划。从创立至今，淘梦都将数据置于核心地位，在这个内容平台的最前端就是数据，淘梦大数据机制是淘梦自主研发的一套数据整理和分析系统，是C2F模式的具体运作，C2F模式指的就是将通过爬虫技术获取视频、社交、媒体、搜索引擎等平台相关数据，并将上千部电影作品标签化，利用数据挖掘及机器学习技术从播放趋势、媒体热度、用户画像、舆情导向、受众反馈等方面对演员、题材、影片等关键词进行挖掘与数据分析。从而选择具有开发潜力的题材，提供给适合的导演与团队开发创作。

这套系统从今年初开始研发，目前还在改进和完善当中，没有对外开放，据阴超介绍，之后准备把这套系统进行产品化运营和管理。

数据之外，淘梦也很注重人才抓取和人才培养，今年陆续推出“5+5导演计划”、“星火计划”，“5+5导演计划”是由五位资深老导演与五位年轻的新导演合作拍摄电影，目前曾志伟、林国华、孙树培等导演都已加入该项计划，“星火计划”脱胎于“5+5导演计划”，淘梦将为青年导演供优质项目策划、资金、制作等全方位、全渠道的一站式支持，建立完整的人才孵化闭环。

![Image](http://si1.go2yd.com/get-image/0IHbBdxwMPQ)

在阴超看来，谁培养了新一代年轻导演就是掌握了中国电影的未来，院线电影公司制片和培养人才的成本都很高，而淘梦可以提供三五百万甚至一千万预算给一个新导演去拍网大，而且这个电影也未必会赔钱。今年淘梦推出的《降龙大师》就属于“5+5导演计划”中的项目，影片由新锐导演项秋良、项河生联合执导，成本300万左右，目前票房收入已经达到1600万。这些经过选拨进入人才培养计划中的青年导演们也会成为淘梦在产业布局重要的一环，淘梦会通过入股公司或工作室、开设工作室等不同合作方式帮助青年导演进行接下来的影视项目创作，淘梦内容平台上的人才和资源也都将得到更进一步扩充。

![Image](http://si1.go2yd.com/get-image/0IHbBfMMP6O)

淘梦创始人&CEO 阴超

将数据与人才串联起来的是淘梦工业化体系制定制片流程，阴超解释说电影是一个很流程化的事情，淘梦目前也在力求做出没有短板的工业化体系，包括通过IT系统指导和保证生产线的流畅与稳定，如高效执行力、完美的预算控制、多元制作风格、多项目的并行管理以及严谨排期等。同时，淘梦遵循“制片人制”，以公司签约的五位核心制片人组建五个工作室，大制片人主要负责筹划和搭建项目，导演把控影片质量和内容，还有许多小制片人和执行制片人进行具体的工作执行。

基于数字内容创新平台，淘梦未来仍将以网剧和网络大电影为重点业务，网剧方面，目前有腾讯动漫IP改编的青春校园剧《腾空之约》，与本山传媒合作的《奉天大饭店》，以及由国漫改编、投资四千万的超级网剧《墓王之王》，三部网剧均会在明年上线。

2.2017年网大市场洗牌，优质玩家业绩拔高顺流而上

众所周知，网大从今年已经进入更新的发展阶段，经历去年的野蛮生长、泥沙俱下，网大从量变开始发生质变，虽然市场产品数量减少、增速减缓，但是更多优质型、创新型产品增多，这是市场经济发展中必经的一环，而这一环也是对之前杂芜的网大环境的一次大浪淘沙，留下的才是真金。

在骨朵前不久公布的Q3网大季报中，网大数量从去年下半年到今年上半年每季度下降10%左右，每季度平均单片贡献的前台播放量趋于平缓，同比去年Q3新增网大播放量下降了三分之二，季度新增网大前台播放量也走势一般，总体而言“平缓”是Q3网大市场乃至今年网大市场所展现的共性，无论是资金投入、项目开发还是公司运作相较去年都冷静和理智许多，去年发热的网大市场今年冷却不少，思量和计算被更多也更重要地纳入网大的每个环节，而算账是淘梦最擅长的，无论是数据研发还是依靠经验对于项目的精准判断，阴超很清楚自己要做一个盈利的公司，赚钱才是生意的最终目的，也正是因为淘梦时刻都在精心规划自己的每一步，才在无论是淘汰赛还是晋级赛中都稳立于不败之地。

![Image](http://si1.go2yd.com/get-image/0IHbBhhL0SG)

在爱奇艺公布的2017年上半年网大票房分账榜单上，淘梦的分账金额占到40%，以8000万位居第一，淘梦出品的《血战铜锣湾3》、《小小爸爸》也连续两月占据爱奇艺网大票房榜榜首位置，同时，阴超在采访中也告诉小娱，淘梦在第四季度初就已经完成全年业绩目标，预计全年收入可达到1.2亿，相较去年翻了一倍，公司利润则已经超过IPO标准，其中自产项目盈利占总盈利的60%以上。

淘梦不仅掌握着行业内头部内容，也拥有极其广泛的资源规模，包括与北京文化、中南影业、英皇、芒果TV等影视巨头达成长期战略合作。同时，业内名导和明星也与淘梦建立了多种形式的合作，如曾志伟出任淘梦“首席内容官”及“首席顾问团主席”，曾志伟、林国华、孙树培加盟淘梦的人才培养计划，钱国伟、张敏担任淘梦监制。赵文卓担任出品人、总监制并主演的“黄飞鸿”系列之《黄飞鸿之生化人》正在后期制作中，陈浩民在与淘梦合作的《斗战胜佛》之后，又一新的合作项目也已启动。纵观淘梦的行业资源与合作对象，无疑是业内首当其中的大牌了。

所谓时势造英雄，在无序的市场中冒头不能完全算是真本事，而在有序健康的环境里步步高升，那可说得上是强手，淘梦的发展几乎与中国网大的发展同步进行，入局早、资历深是淘梦的第一优势，但在高手林立的市场中站稳脚跟，凭借的就是淘梦精打细算、互联网思维灵活运用的核心优势，阴超也提到——“我们通过数据看到我们的用户是谁，他们希望看到什么样的产品，我们从而生产和提供相对应的产品”。

从商业角度来说，最打动人心的产品无非就是最贴心和最实用的产品，如淘梦的《道士出山》系列、《国产大英雄》系列、《血战铜锣湾》系列，同时《海带》在豆瓣上获得7.2分，这在网大豆瓣3-4分的平均分中算得上绝对高分了，从这一现象也可见，淘梦已然成为网大行业引领优质内容的主力军之一。

淘梦成立至今经历过四次转型，今年初淘梦成立淘梦控股公司，旗下包括淘梦网络、淘梦影业、淘梦华元、美亚淘梦（香港）、淘梦文学、淘梦娱乐、淘梦传播、铠甲娱乐等全资或控股子公司，阴超表示，“这次也算是一个新的前进和定位，是公司一个升级式的发展，所以我们定义自己是具有创新力的内容平台。”。

总体而言，以网大网剧为代表的中国数字内容市场仍旧处于发展早期阶段，虽然经历去年监管政策的洗牌和今年的重整市场规范许多，但未来发展还有诸多可能性，不过从淘梦身上我们似乎也看到一条以不变应万变的制胜定律，那便是低调踏实地做好内容，相信在未来淘梦还将为市场和观众“淘”出更多惊喜。
