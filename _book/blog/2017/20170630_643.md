# 24处禁令新升级！我们发现网络禁令比电视端还严格！

*陈梦茹 刘燕秋|2017-06-30|电影 
                                                电视剧*

恐怕以后再也不能说，网生内容的监管没有明确的边界了。

就在今天（6月30日），中国网络视听节目服务协会发布《网络视听节目内容审查通则》（以下简称《网络通则》）。

这似乎有点“一锤定音”的意味在。网生内容监管的风声在近期越吹越大，仅从6月份来看，短短30天，网信办、文化部和广电总局等部门就相继出台了将近10个文件。

![Image](https://mmbiz.qlogo.cn/mmbiz_png/jNZszpkibXxib7fwwJpDNgQ6ExPOGhIWXjvphthTHYtFu4obQibkmzjumicdicVpJgpBKnIuIrmZXWOlAOiaTURqk4tQ/0?wx_fmt=png)

数十个八卦大号的新浪微博账号突然被封；近百个娱乐微信公众号被封；ACFUN、新浪微博等视听节目被封，再到昨日文化部关停12家网络表演平台，都无一不在预示着这场“最严监管”的强势袭来。

而《网络通则》的出台，也让网生内容从业者意识到，严管是真的来了。

![Image](https://mmbiz.qlogo.cn/mmbiz_png/jNZszpkibXxib7fwwJpDNgQ6ExPOGhIWXj3rRf2FpwDkXhXuK41fgwfJb17PFnptln3Aiajb2aurM7Z6za5f0ZnCQ/0?wx_fmt=png)

很有意思的是，河豚君还对比了一下《网络视听节目内容审核通则》和《电视剧内容制作通则》（以下简称《电视通则》）。娱乐资本论发现，网络视听节目内容的审查比电视剧内容制作的监管更加详细，也更加严格。具体来说，在9处地方，有了升级；在15处，有了新增，可以说是足足严格了“24处”。

要来的，总比想象中更猛烈一些？

1.相比电视剧，网络视听节目内容审查更加严格

所谓的网络比电视更严，其中主要体现在对未成年人造成影响的视频内容，比如恶搞表现形式的视听内容和露骨表现“性”的内容；关于宗教信仰和明星炒作的内容也都有更多细节的提及。

第一，是在导向要求上：

1.《电视通则》第四条中要求大力弘扬中国优秀传统文化。《网络通则》则在此基础上增加了“深入阐发讲仁爱、重民本、守诚信、崇正义、尚和合、求大同等核心思想理念。”

2.《网络通则》第五条中增加了“持续颂扬自强不息、敬业乐群、扶危济困、见义勇为、孝老爱亲等传统美德。”

第二，在规定不得制作的内容中：

1.在危害国家统一的条例上，《网络通则》增加了不得“宣扬恐怖主义的内容”。

第三，在规定不得出现的内容中：

关于不符合国情和社会制度，有损国家形象，危害国家统一和社会稳定中：

1.《网络通则》第2点增加了“贬损、恶搞、损害革命领袖、英雄人物的形象、名誉”。

关于违背国家宗教政策：

1.《网络通则》第4点增加了“歪曲、诋毁或歧视宗教观念、宗教信仰和宗教称谓”。

关于渲染淫秽色情和庸俗低级趣味：

1.在《电视通则》中规定不得“展示和宣扬不健康的婚恋观和婚恋状态，如婚外恋、一夜情、性自由等”中，《网络通则》增加了“换妻”。

关于危害社会公德，对未成年人造成不良影响的：

1.《网络通则》第1点增加了“以恶搞方式描绘重大自然灾害、意外事故、恐怖事件、战争等灾难场面”；

2.第2点增加了“以肯定、赞许的基调或引入模仿的方式表现打架斗殴、羞辱他人、污言秽语等”；

3.第6点“展示未成年人或者未成年人形象的动画、动漫人物的性行为等”。

值得注意的是，《通则》出现一条了全新的规定，除了上述规定外，还有规定在网络视听节目中不能出现的内容：

![Image](https://mmbiz.qlogo.cn/mmbiz_png/jNZszpkibXxib7fwwJpDNgQ6ExPOGhIWXjGiaVKm14NqFIFzcSuq3n0XPYy7df7Iiaua7gNK9cfu0Dduhv8RxyWNHA/0?wx_fmt=png)

这主要还是针对近年来在网络上盛行的视频内容。比如以明星绯闻炒作的、诱导未成年人谈论名利或者为了吸引眼球刻意展现丑态的视频内容，都不能再出现。

第四，在审核原则上：

电视剧的审核一直是由总局把控，而视频网站的视听内容一直以来都是自审自查的，所以在网络视听节目的审查通则中，第二章就明确提出了审核原则，其中包括具体的审核要素、审核要坚持先审后播和审核到位的原则。

娱乐资本论曾了解到，每一家视频网站都有自己专门的审查部门，它们根据自家平台的实际情况来成立审查小组。而广电总局会派专门的人员来对这些视频平台的审核小组进行专门的培训，审核人员都要持证上岗。

其实娱乐资本论之所以对比这两个《通则》，一方面都是行业协会推出，另一方面也都在2016-2017这同一个周期内，两者具有相当大的可比性。

2.网剧网大受波及最大，同性题材是关注重点

过去一年，网剧、网大和网络综艺都不断受到政策影响，遭遇被下架或者整改删减再上架的命运。

按照时间顺序来看，从去年的《偶像就该酱婶》，到今年年初的《姐姐好饿》、《黑白星球》和《hello，女神》都被悄然下架，原因不乏污段子太多、性话题讨论尺度过大等原因，这也和《通则》列出的审核标准中需要删除的内容一致。

网剧和网大则受到更大影响。以去年的爆款网剧《余罪》来说，在播出很长一段时间之后，也因“剧中人设不符合警察形象”等原因被下架。而网剧《十宗罪》，据悉也在临近上线前期被要求整改，删除较多涉及血腥暴利的镜头之后才最终得以上线。

再看今年2月7日，腾讯视频发布公告，称腾讯视频PGC影视组对上线网络剧进行排查，其中107部网络剧由于未在片头添加网络视听节目信息备案编号，不满足网络视听节目上线条件，采取集中下架处理。

6月20日，40多部网大突然被下架，其中包括《二龙湖浩哥》全系列、《我的室友是狐仙》和《打狼之我命由己》等热度较高的作品。在此之前，声响更大的一次网大下架行动则是在2016年11月，有60多部网大作品在几天内被相继下线。

网剧方面，表现同性之间情感的网络剧一直是被政策盯得最紧的领域。不说去年被集中下线的几部网大了，就说最近由《梁山伯与祝英台》改编的网络剧，就经历了一段波折。在去年举办的新闻发布会上，该改编网络剧公布了主演阵容，而饰演祝英台一角的却是一位男性演员，也被网友称为“最美祝英台”。

但是好景不长，该剧在开拍不久就遭遇了一系列“疑似审查”的问题，比如中途换演员，将饰演祝英台的男性演员替换成了女性演员。据悉，在拍摄完成后，这部剧还面临被要求改剧名。

巧合的是，今日，德国议会投票批准同性婚姻。

除此之外，从《通则》中可以看出，涉及“宣扬灵魂附体、转世轮回、巫术作法等封建迷信思想”和“表现离奇、怪诞的犯罪案件”的视频内容都要进行删减之间才能上线。有网友评论，这意味着“玄幻类IP和破案类IP都要倒下了。”

其实如果真要严格按照此项规定，不但是以上的剧集，像四大名著中，《西游记》也会涉及“转世轮回”等封建迷信，《三国演义》也会涉及“渲染暴力”，《红楼梦》会“展现非正常的性关系”，《水浒传》更是从“宣扬以暴易暴”到“展现非正常的性关系”等碰触了个遍。

3.在新政之下，相关各方该如何愉快地玩耍？

事实上，《通则》一出，网络上反对的声音也有不少，都认为这样的细则等于让以后无剧可播了。

在刚过去的上影节上，腾讯、优酷和爱奇艺等视频平台，光线、芒果娱乐等影视公司都公布了不少新片单。如果真的要受到新政的严格监管，《SCI迷案集》、《超隐形男友》等会不会将受到影响？因为这些剧集内容都“擦边”了恐怖、迷信等情节。

而从名字上就看得出涉及鬼怪、犯罪等剧集就更不能幸免，例如《鬼吹灯之牧野诡事》、《余罪》第三部和第四部以及漫改剧《灵域》等近10部作品，里面都大量涉及《通则》中所提及的暴力、封建迷信等元素。《余罪》前两部更是在此前就遭遇了下架。

但说回来，经历过一系列的政策影响之后，视频平台也学会了“上有政策，下有对策。”在年初政策收紧风声频出的时候，视频平台也出台了相应的审片标准。比如“爱奇艺九禁“、“乐视十不准”和“搜狐七把控”，都是为了响应政策净化网络环境的号召。

除此之外，视频平台还专门由专人来负责跟政府沟通。例如，去年11月，为了响应广电总局让商业视频网站创办自审自播团队的号召，爱奇艺成立了网络影视节目规划研究中心，主要负责监测、把关视频网站自制或引进内容。B站也从SMG挖来专人负责政府关系，近几年非常积极努力扮演为95后、00后年轻人代言、倡导主流价值观的角色。

更别提爱奇艺的很多高管都是来自体制内，已经担任爱奇艺首席内容官的王晓晖就是前中央人民广播电台副台长，而刚刚宣布离开优酷的王平则是湖南卫视副台长，这些深谙体制内规则的人自然能做到与监管部门沟通畅通。

除了视频平台寻求与监管部门的沟通，以防踩了红线，很多有嫌疑踩红线的剧集也积极的寻找应对之策。例如电影《烈日灼心》就在改编过程中，对原著中男主角“辛小丰”的同性恋情节进行了删减。《嫁给大山的女人》则把“被拐卖到大山深处的女人杀死了全村人”改成了“最后她被警察解救”。大头武媚娘就是在被要求整改之后，美艳的妃子们都变成了大头贴。网综《奇葩说》初期因为马东这位老司机“飙车”，再加上有一期涉及“同志”话题被要求下架整改。而现在《奇葩说》涉及到脏字都用“哔”代替，而辩题也开始寻求安全落地。

媒体专家魏武挥表示，这次视听新政不能是说越来越严了，是本来就很严，“只是把监管条文化、公开化，让视频平台有了非常明确的操作细则”。同时他也说到，“目前大家都不能做了，没有了差异性区别对待。而且视听协会是一个行业自律组织，说白了就是一个倡议。”

据娱乐资本论了解，因为此次发布的中国网络视听节目服务协会只是官方机构下辖的一个协会，所以大家才会非常紧张。但正式的规定仍得参照官方今年6月1日印发的《关于进一步加强网络视听节目创作播出管理的通知》。

从实操层面来说，《通则》中建议不要触及的具体内容很难规避。像性自由，任何家庭剧只要主角婚前有性行为，都可以划归？再比如未成年人早恋，青梅竹马两小无猜算不算？如果纯洁的童年情谊硬是被安上早恋的帽子，这也是无稽之谈。

在娱乐资本论看来，目前不少网剧，即时有些题材的“擦边”，但具体分析，其立意好，故事佳，整体价值观倾向是健康的（像《法医秦明》这样展现我国法医优秀的破案能力和坚定的职业精神的），为什么要被干涉呢？

行业协会提倡自律是好事，但自律不等于把自己管死。而应该体现在行业的方方面面，比如购销潜规则、点击量造假等，这些暗流涌动破坏行业信用的行为更需要监管。
