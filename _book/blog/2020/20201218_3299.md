# AppAnnie大中华区负责人戴彬 ：抓住成长期中的Z世代

*娱乐资本论|2020-12-18|大公司 
                                                科技*

12月15日，以「增长新引擎」为主题的深响年度机遇峰会在北京举行。App Annie大中华区负责人戴彬在会上做了“制胜Z世代——Z世代移动行为分析”的演讲，基于App Annie的数据分享了2021年移动行业将呈现的五大趋势。

演讲中，戴彬具体分享了五个方面的预测：2021年，Tiktok的平均月活跃用户数将达到12亿；居家活动将实现 43%的加速增长；移动游戏领域的用户支出将超过1200亿美元；移动广告支出将达到2900亿美元；与疫情之前的水平相比，消费者安装的在线视频应用数量将增加85%。

戴彬表示，“我们常听到的Z世代就是我们所说的95后，数据分析的角度是16-24岁这部分的用户，这个世代的喜好，会很大程度上影响到未来的市场。”

Z世代是移动的一代，戴彬在深响年度机遇峰会上对Z世代的行为特征做了如下判断：Z世代对于新兴市场的影响力更为显著；Z世代移动用户的崛起速度更快；Z世代在日常生活中与移动互联网的联络更为紧密。

戴彬表示：“在娱乐方面，我们看到Z世代对于视觉体验、对于移动、对于社交和游戏融合方面的重视。我们从购物和财务方面也看到，他们打开购物APP，不一定要去购物，而是为了获得他们想要的体验和他们所关注的内容。”

在Z世代的成长期接触他们，有助于将客户生命周期价值最大化。

以下为演讲全文：

各位好，我是AppAnnie的戴彬。对于做互联网出海的人来说，AppAnnie这个名字很熟悉。我们从2010年开始就在做全球的APP市场数据监测，帮助用户来了解市场的变化、趋势是什么、商机在哪里等问题。

今天来到「深响」的舞台，AppAnnie要给大家带来什么样的启发呢？刚才我就在想这个问题，虽然我们来自于不同的行业，有不同的业务，但是有一个共同点：我们都很关注用户，关注人在发生着什么样的变化，或者从整体来看趋势是什么。

通过对全球APP数据的分析，今天我和大家一起来看一下全球的市场和消费者，特别是年轻的消费者他们的兴趣点在如何转向。接下来我分享的内容都是基于AppAnnie的全球APP市场数据进行的估算。

首先是AppAnnie最近发布的对2020年后疫情时代数字世界新常态的预测，主要包括五个方面。

第一，结合今年的疫情状况，一些领域的线上APP增长非常明显，而且这个增长会持续到明年。不管疫情之后发生怎样的情况，这些增长已经被推到了前沿的位置。这些领域有教育、商务，同时也包括在线视频、购物和财务，还有送餐、外卖、健身等等。

我想提出几个点，一个是在线视频有比较快的增长，达到38%；消费者手机上安装的APP数量也有明显的增长。2019年美国、韩国和英国市场平均有四五个在线视频APP，从今年的趋势来预测，明年在线视频还会有一个大幅的增长。

这个预测给我们带来一个什么样的启发呢？我认为视频服务在往多元化的方向发展，我们不会看到用户只集中在几个超级APP上，消费者还会有更多的选项。

第二，是最近特别火的TikTok抖音海外版，我们对它的预测是2020年全球活跃用户会达到12亿的数量级。他们目前的实际数据也已经超过了10亿，进入了全球月活10亿级APP俱乐部。如此体量的案例其实不多，屏幕上有Facebook，也有微信。进入到这一数量级的APP，意味着它已经是社交品类当中非常重要的一环。

这个数据的预测给我们带来什么样的启发呢？我们需要看一下Tiktok和传统十亿级社交APP有什么不一样，特别是Facebook。

我们往往认为TikTok是短视频APP，但现在我们要把它划分到社交这一类别，和有巨大体量的社交类APP做类比。TikTok的信息载体，包括消费的内容都是视频内容，亚澜开场时讲图文内容推广有一些吃力，很多消费者在进行娱乐化或者是社交化消费时，他的诉求都在短视频上，而且是在不断碎片化的。所以多样化和碎片化，是我们对未来两个方向的预期。

第三，游戏APP在全球的营收、用户支出和广告营收方面都有增长的趋势。这样的趋势告诉我们，整个线上经济还是非常值得我们继续关注的。问题就在于怎样去关注，哪些方面是它的机会所在。

接下来我想从我们对Z世代数据的分析角度来和大家一起探索。所谓Z世代，就是我们所说的95后，数据分析的角度是16-24岁这部分的用户，现在来专门分析他们在用APP时，表现出来的喜好偏向是什么。

首先我想通过全球的数据观察，来看看在全球范围之内Z世代表现出的趋向。像印尼这样的新兴市场，Z世代比例非常高，将近一半活跃用户的设备来自于Z世代。在日韩等更加成熟的市场，因为互联网普及，活跃用户在各年龄段都很多，所以Z世代的比例并不是很高，但Z世代的增长是更快的。

印尼、巴西这些新兴市场，Z世代的增速非常快，在韩国，Z世代的增长是其它年龄段用户增长的两倍。Z世代在新兴市场已经占了很大的比例，成熟市场这些活跃用户的增速更快，这个世代的喜好，会很大程度上影响到未来的市场。

他们具体表现是怎样的呢？简单说结论，我们会发现，Z世代的用户会把部分时间从游戏上拿出来，放在非游戏领域。和25岁以上的用户相比，他们在非游戏的APP上分配的时间比重更大。对于希望通过线上、特别是通过APP来和Z世代用户互动的各行业来说，这是一个好消息。

另外一点，我们发现Z世代在使用手机时，他们碎片化程度更高，平均每次花在一款APP或一款游戏上的时间相对更短。他们更加频繁地在不同应用、不同场景内切换。这告诉我们，他们注意力集中的时间更短，跳跃的次数更多，应用要抓住他们注意力的难度更大，所以需要更加清楚地理解他们的诉求、喜好，才能够更好的跟他们互动。

Z世代更关注什么，用APP的时候他们就集中在哪里。答案其实就是漫画、社交和游戏，其中漫画对Z世代来说是比较重要的娱乐方式。漫画、社交和游戏是Z世代在线上最重要的三个领域，这需要我们去关注。

Z世代关注的第二个梯队APP，包括教育、其它的娱乐和摄影摄像等等。年轻人不是那么喜欢的，包括财务、出行、天气，甚至也有购物。这个就需要我们去思考：对于这样的用户诉求，我们怎么去理解和把握；如果我在做这方面的业务，怎样去顺应年轻一代他们对线上体验的诉求。

讲到社交，给大家看一个榜单。这个是头部APP的Z世代指数排行榜，Z世代比例越高、指数越大，那么它的排名就越高。在海外不管是欧美，还是东南亚国家一般排在前面的都是Facebook、Twitter，但是这个用Z世代指数来排的榜单就不一样了。

![Image](https://p1-tt-ipv6.byteimg.com/origin/pgc-image/8d3e295bf9c246dda049747c1952f7fd)

可以看到，排在最前面的是Tiktok。刚才我们讲到它颠覆了传统以图文为主流的社交体验，给到了年轻消费者不一样的线上体验，所以它在这些国家中排在了前面，这也印证了我们对之前说到的那几点年轻消费者偏好趋势。

法国是Tiktok唯一没有排到第一的市场。法国市场的第一名是Discord，这个APP对我们来说比较小众、不太熟悉，但在Z世代面前不是如此。Discord是一个游戏社交平台，为什么一个游戏社交平台如此重要？根据我们的一些思考和研究发现，对Z世代来讲游戏、社交、娱乐等概念已开始融合，Discord的用户虽然是游戏用户，但他们在Discord上和朋友互动的主题不只包括游戏。另外，Discord上大量表情包的使用，也印证了视觉体验对于Z世代在社交方面的诉求。

其次是娱乐和视频。国内讲到视频应用，最熟悉的是腾讯视频、优酷、爱奇艺等APP，但在海外基本上就是Netflix或者YouTube。但对Z世代来讲，他们喜欢的视频和娱乐也是不一样的。他们不太像我们想象的那样去追剧，而是关注像Twitch这样的游戏直播平台，如日本的niconico，一个相当于日本的B站、有弹幕的老牌网站；还有在印尼、韩国和美国流行的Webtoon，用中文讲就是网络漫画，是为手机量身定制的漫画书和内容，读者在读的时候直接划，内容会源源不断出来。

另外，视频广告成为了获取Z世代的有效形式，这体现在视频广告的支出在增加，视频广告确实契合了Z世代对视觉体验的喜爱。

在财务和购物两类APP中，Z世代的比例比较低，但从增长来看，Z世代的增速非常快，超过了60%。虽然Z世代目前量不多，但是它的增长速度非常快，未来会很快占有市场。

再来看一下购物。在座的很多朋友可能是和购物相关的行业，我们看Z世代在打开一个购物APP时，他们的诉求是什么。我们从图上大概可以看出一些端倪。首先排到前面的，特别是蓝色部分，没有刚才娱乐和社交那么多，这些电商APP都是年轻人喜欢的时尚潮流，买喜欢的潮牌服装或者是美妆的地方，这切中了Z世代在购物领域的诉求。

![Image](https://p1-tt-ipv6.byteimg.com/origin/pgc-image/3a1e50c2ed7a4b449989f3a0046c1ad4)

最后在财务方面，财务APP里面Z世代的比例最低。这些APP以国外的银行为主，是传统银行做线上APP的一些服务。在全球范围内，很多线下的资源渠道，包括银行确实开始意识到这个问题——要跟用户互动，大家发布APP就是希望把用户转化到线上。因此需要了解的是，对于Z世代来说，他们希望的互动方式是怎样的，之后我们的互动才可以满足他们，契合他们的需求。

在娱乐方面，我们看到Z世代对于视觉体验、移动、社交和游戏融合方面的重视。我们从购物和财务方面也看到，他们打开购物APP不一定要去购物，而是为了获得他们想要的体验和他们所关注的内容。
